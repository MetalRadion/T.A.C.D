The Automatic Cat Door

Un poco sobre nosotros

T.A.C.D tiene como objetivo la correcta recolección de datos sobre la actividad de nuestro amigo felino favorito para poder ser presentada de manera útil y comprensible para el usuario, al mismo tiempo T.A.C.D cuenta con funciones automatizadas para la operación de una trampilla asociada con nuestro software la cual habilita que nuestra mascota sea capaz de tomar un relajante paseo.
T.A.C.D es desarrollado y patentado por Benjamín Olivares, Christopher O'Kinggton y Lucas Rojas.
